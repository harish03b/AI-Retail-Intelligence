# SQLAlchemy ORM Models

## Objective

Create Python ORM models that map directly to the retail data warehouse tables.

## Models

- Customer → dim_customer
- Category → dim_category
- Product → dim_product
- Store → dim_store
- DateDimension → dim_date
- FactSales → fact_sales

## Benefits

- Type-safe database operations
- ORM-based inserts and queries
- Easier maintenance
- Supports repository pattern
- Production-ready architecture
